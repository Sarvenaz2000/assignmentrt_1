Detailed Description

User interface node for controlling turtlesim turtles.

Author
Sarvenaz Ashoori
Version
0.1
Date
21/02/2025

This node allows the user to manually control two turtles in the turtlesim environment. It provides a menu-based interface for selecting a turtle and specifying movement commands.

Services Used:

  • /spawn (turtlesim::Spawn): Service to spawn a new turtle.

Published Topics:

  • /turtle1/cmd_vel (geometry_msgs::Twist): Velocity command for turtle1.
  • /turtle2/cmd_vel (geometry_msgs::Twist): Velocity command for turtle2.
